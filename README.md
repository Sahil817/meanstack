HTML and CSS tags used in the MEAN_ASS_1 :

HTML tags :

| Tag               | Description                               |
| ----------------- | ----------------------------------------- |
| `<!DOCTYPE html>` | Declares HTML5 document type.             |
| `<html>`          | Root element of the webpage.              |
| `<head>`          | Contains meta data, title, links, styles. |
| `<meta>`          | Provides page info (charset, viewport).   |
| `<title>`         | Displays page title in browser tab.       |
| `<link>`          | Used to link CSS file or fonts.           |
| `<body>`          | Contains all visible webpage content.     |
| `<div>`           | General container used to group content.  |
| `<aside>`         | Sidebar section of the page.              |
| `<nav>`           | Navigation menu container.                |
| `<a>`             | Hyperlink (menu items).                   |
| `<span>`          | Inline text container (icons, labels).    |
| `<main>`          | Main content area of page.                |
| `<header>`        | Top header section.                       |
| `<input>`         | Used for text fields like Search.         |
| `<section>`       | Groups related content blocks.            |
| `<h1> - <h3>`     | Heading tags for titles.                  |
| `<select>`        | Dropdown menu.                            |
| `<option>`        | Dropdown item inside select.              |
| `<table>`         | Creates table structure.                  |
| `<thead>`         | Table header section.                     |
| `<tbody>`         | Table body rows.                          |
| `<tr>`            | Table row.                                |
| `<th>`            | Header cell in table.                     |
| `<td>`            | Data cell in table.                       |
| `<form>`          | Form container for input fields.          |
| `<label>`         | Text label for form fields.               |
| `<button>`        | Clickable button.                         |
| `<footer>`        | Bottom section of page.                   |

  CSS tags :

  | Selector | Description                              |
| -------- | ---------------------------------------- |
| `*`      | Selects all elements for global styling. |
| `body`   | Styles page background, layout.          |
| `.class` | Styles elements with class attribute.    |
| `#id`    | Styles elements with unique ID.          |
| Property                | Description                          |
| ----------------------- | ------------------------------------ |
| `display: flex;`        | Align items horizontally/vertically. |
| `flex-direction`        | Row or column layout.                |
| `justify-content`       | Space distribution horizontally.     |
| `align-items`           | Align items vertically.              |
| `grid-template-columns` | Creates responsive grid layouts.     |
| `gap`                   | Space between grid/flex items.       |
| Property        | Description                                 |
| --------------- | ------------------------------------------- |
| `margin`        | Space outside element.                      |
| `padding`       | Space inside element.                       |
| `border`        | Outline around elements.                    |
| `border-radius` | Makes corners rounded.                      |
| `box-shadow`    | Adds shadow behind boxes.                   |
| `box-sizing`    | Controls how width & height are calculated. |
| Property           | Description                        |
| ------------------ | ---------------------------------- |
| `background`       | Sets background color or gradient. |
| `background-color` | Solid background color.            |
| `opacity`          | Transparency of an element.        |
| Property           | Description                    |
| ------------------ | ------------------------------ |
| `cursor: pointer;` | Shows hand pointer on hover.   |
| `transition`       | Smooth animations on hover.    |
| `.badge`           | Styled label for stock status. |
